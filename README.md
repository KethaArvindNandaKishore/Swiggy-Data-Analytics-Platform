# 📊 Swiggy Data Analytics Platform

An end-to-end Business Intelligence and Data Analytics project built using **Microsoft Fabric**, **SQL**, and **Power BI** to analyze Swiggy's food delivery data. The project focuses on designing a modern analytics platform that transforms raw operational data into actionable business insights through automated ETL pipelines, interactive dashboards, and performance reporting.

---

## 🚀 Project Overview

The objective of this project is to build a scalable analytics platform capable of processing large volumes of customer and operational data while enabling data-driven business decisions.

Using Microsoft Fabric, data is extracted, cleaned, transformed, and stored in a structured analytics model. SQL is used to analyze customer behavior and business performance, while Power BI provides interactive dashboards for monitoring KPIs and identifying growth opportunities.
